# Mechafind

Mechafind is a full-stack web application built with **React + Vite** (frontend) and **Node.js + Express.js** (backend). It provides a platform for finding and managing mechanical-related resources.

## Project Structure

```
Mechafind/
├── backend/          # Express.js API server
│   ├── controllers/  # Request handlers
│   ├── middleware/   # Custom middleware
│   ├── models/       # Database models
│   ├── routes/       # API routes
│   └── server.js     # Entry point
├── frontend/         # React + Vite frontend
│   ├── src/          # React source files
│   ├── public/       # Static assets
│   └── vite.config.js
└── .gitignore
```

## Tech Stack

- **Frontend**: React, Vite, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (via Mongoose)

## Getting Started

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
